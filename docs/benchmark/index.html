Verify It Yourself

Verify any published proof bundle independently. No proprietary software required.

Quick Start

1. Install
pip install proofcodec-verify
2. Verify a proof bundle
proofcodec-verify bundle path/to/kqvk_proof/
3. Check compression claims
proofcodec-verify baselines 144508 23048 200896 --model-bits 219
4. Decode a residual file
proofcodec-verify decode path/to/residual.v18
